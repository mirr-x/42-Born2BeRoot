# Prompt for Creating a Classic Hacker-Style README

Use this prompt with any AI to generate a beautiful old-school hacker aesthetic README:

---

## The Prompt

```
Create a beautiful old-school hacker-style README.md for my Born2beRoot project with the following characteristics:

**Classic Hacker Aesthetic:**
- ASCII art banner/logo at the top (creative terminal-style art)
- Retro terminal/console visual elements
- Use of `┌─`, `└─`, `│`, `─`, `┬`, `├`, `┤` box drawing characters
- Monospace code blocks with terminal-style prompts ($ or #)
- "Matrix-style" or cyberpunk visual elements
- Green/cyan color mentions in descriptions
- Retro computer/hacking terminology

**Content Structure:**
- ASCII art title banner
- Brief "system information" styled description
- Project overview presented like a system log or terminal output
- Installation instructions as terminal commands with proper prompts
- Features list with checkbox style [✓] or [x] markers
- Tech stack displayed as "loaded modules" or "dependencies"
- Security features highlighted with lock/key symbols
- Usage examples in realistic terminal session format
- Credits/footer with ASCII art or signature

**Visual Style:**
- Clean, organized sections separated by ASCII borders
- Creative use of emojis: 🔒 🔑 🖥️ ⚡ 🛡️ 💀 👾 🕹️
- Terminal-style color indicators (even if just mentioned)
- Retro computer specs format for technical details
- "Loading..." or progress bar ASCII art elements
- Glitch-style or stylized text for section headers

**Content to Include:**
- Project: Born2beRoot (42 School system administration)
- Virtual machine setup with Debian
- Security features: encrypted LVM, AppArmor, UFW firewall, SSH
- Monitoring script that runs via cron
- User/sudo management
- Technology comparisons (Debian vs Rocky, AppArmor vs SELinux, etc.)

Make it look like something from the 90s hacker culture but clean and professional. Think: old BBS systems, classic terminal UIs, retro computing aesthetics mixed with modern GitHub markdown capabilities.
```

## Tips for Best Results

1. Be specific about wanting "retro terminal aesthetic"
2. Mention specific ASCII art styles you like
3. Ask for creative use of box-drawing characters
4. Request terminal-style command examples
5. Specify emoji usage for visual appeal
6. Ask for both style AND substance in content

## Quick Usage

Copy the prompt above and paste it into:
- ChatGPT
- Claude
- Any AI that can generate markdown
- GitHub Copilot Chat

The AI will generate a complete hacker-style README that you can use directly or customize further.
